# global-sales-power bi-dashboard
 Power BI dashboard analyzing global sales performanc

# Project Overview
This project analyzes global sales performance using Power BI.  
The dashboard provides insights into sales, profit, quantity, customer performance, and geographic trends.

# Business Questions Answered
- What is the total sales and profit performance?
- Which customers generate the highest revenue?
- How do sales and profit trend over time?
- Which regions contribute the most to global sales?
- What is the profit margin across segments?

# Tools & Technologies
- Power BI Desktop
- DAX (Measures & KPIs)
- Data Modeling
- Data Visualization Best Practices

# Key Metrics
- Total Sales
- Total Profit
- Total Quantity Sold
- Profit Margin %
